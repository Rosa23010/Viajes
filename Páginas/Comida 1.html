Datos de Interés

Red Informática

Una red informática es un conjunto de dispositivos (como ordenadores, servidores e impresoras) conectados entre sí para compartir información, recursos y servicios a través de medios físicos o inalámbricos.

LAN y WAN

Una LAN (Red de Área Local) conecta dispositivos en un espacio reducido, como una casa, oficina o colegio. Una WAN (Red de Área Amplia) conecta redes a gran distancia, como ocurre con Internet.

Proveedor de servicios de Internet

Un proveedor de servicios de Internet (ISP) es la empresa que ofrece acceso a Internet a los usuarios, permitiendo la conexión desde hogares o empresas mediante fibra, ADSL o datos móviles.

Dominio en Internet

Un dominio es el nombre único que identifica a un sitio web en Internet, como por ejemplo www.ejemplo.com, y facilita el acceso sin necesidad de recordar direcciones IP numéricas.

Protocolo informático

Un protocolo informático es un conjunto de normas y reglas que permiten que los dispositivos se comuniquen entre sí de forma correcta dentro de una red.

Dirección IP

Una dirección IP es un número único que identifica a cada dispositivo conectado a una red y permite enviar y recibir información correctamente.

URL

Una URL (Localizador Uniforme de Recursos) es la dirección completa que se escribe en el navegador para acceder a una página web, e incluye el protocolo, el dominio y, en ocasiones, la ruta específica.

Protocolo DNS

El DNS (Sistema de Nombres de Dominio) es el protocolo que traduce los nombres de dominio, como www.ejemplo.com, en direcciones IP para que los navegadores puedan localizar los servidores correctos.

Protocolo HTTP y HTTPS

HTTP es el protocolo que permite la transferencia de información entre el navegador y el servidor web. HTTPS es su versión segura, ya que cifra los datos para proteger la información del usuario.

Página web estática vs página web dinámica

Una página web estática muestra siempre el mismo contenido y no cambia según el usuario. Una página web dinámica genera contenido diferente según la interacción del usuario o la información almacenada en una base de datos.

El alojamiento web/hosting

El alojamiento web o hosting es el servicio que permite almacenar los archivos de una página web en un servidor para que esté disponible en Internet las 24 horas del día.

CMS

Un CMS (Sistema de Gestión de Contenidos) es una herramienta que permite crear, administrar y modificar páginas web sin necesidad de programar desde cero.
